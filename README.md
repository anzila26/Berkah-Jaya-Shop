# Berkah Jaya Shop
Aplikasi penjualan alat-alat listrik berbasis android, pelanggan dapat membeli produk secara online melalui aplikasi dengan mudah tanpa perlu datang langsung ke toko.
## Fitur yang dimiliki :
- login
- regist
- wishlist
- checkout
- history pemesanan
- komplain
- ulasan
- edit profile
